1. What is QA?
Quality Assurance means preventing defects by improving the process of building software.

2. What is a Test Case?
A test case is a step-by-step instruction that describes what to test, how to test it, and what the expected result should be.

3. What is a Bug Report?
A bug report is a document that explains a problem in the software, how to reproduce it, and what the expected behavior should be.
